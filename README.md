Multi_Agent_Project

This repository contains a simple multi-agent system project built for practice and testing.
It includes multiple agents that work together to perform tasks such as data processing, automation, or simple decision-making.

Features

Multiple agents working together

Modular code structure

Easy to modify and extend

Beginner-friendly project layout


How to Use

1. Clone the repository:

git clone https://github.com/hinaanoortariq26-ctrl/multi_agent_project.git


2. Open the project folder on your computer.


3. Run the main script (example):

python main.py



Project Structure

multi_agent_project/
│-- agent1.py
│-- agent2.py
│-- main.py
│-- README.md

Requirements

Install required libraries:

pip install -r requirements.txt
